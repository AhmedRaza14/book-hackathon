# ہفتہ 2: روبوٹکس کے لیے ریاضی کی بنیاد

## سیکھنے کے اہداف
- روبوٹکس کے لیے لکیری الجبرا کے تصورات کو لاگو کریں
- کنیمیٹک ٹرانسفارمیشنز کو سمجھیں
- جیکوبین اور ٹرانسفارمیشن میٹرکس کے ساتھ کام کریں
- فارورڈ اور انورس کنیمیٹکس ایکویشنز کو نافذ کریں

## 2.1 روبوٹکس کے لیے لکیری الجبرا

لکیری الجبرا روبوٹکس کے لیے ریاضی کی بنیاد فراہم کرتا ہے، جس سے ہم جگہ، تبدیلیوں، اور سسٹم ڈائی نامکس کو ظاہر کر سکتے ہیں اور ان کا انتظام کر سکتے ہیں۔

### روبوٹکس میں ویکٹرز

ویکٹرز کو روبوٹکس میں بہت استعمال کیا جاتا ہے تاکہ ظاہر کیا جا سکے:
- 3D جگہ میں پوزیشن اور اورینٹیشن
- رفتار اور تیزی
- قوتیں اور ٹورکس
- جوڑی کے زاویے اور کنفیگریشنز

### روبوٹکس میں میٹرکس

میٹرکس ہمیں ظاہر کرنے کی اجازت دیتے ہیں:
- لکیری ٹرانسفارمیشنز (گھماؤ، اسکیل، مڑنا)
- سسٹم ڈائی نامکس اور کنٹرول میٹرکس
- اطمانان کے لیے کوویرینس میٹرکس
- رفتار کے تعلقات کے لیے جیکوبین میٹرکس

## 2.2 فارورڈ کنیمیٹکس

فارورڈ کنیمیٹکس مسئلہ حل کرتا ہے کہ جوڑی کے زاویوں کو دیکھتے ہوئے اینڈ ایفیکٹر کی پوزیشن اور اورینٹیشن کا تعین کرنا۔

### ٹرانسفارمیشن میٹرکس

 مختلف کوآرڈینیٹ فریموں کے درمیان تعلق ٹرانسفارمیشن میٹرکس کے ذریعے ظاہر کیا جاتا ہے۔ ایک 4x4 ٹرانسفارمیشن میٹرکس $T$ گھماؤ $R$ اور ترجمہ $p$ کو جوڑتا ہے:

$$T = \begin{bmatrix} R & p \\ 0^T & 1 \end{bmatrix}$$

جہاں $R$ ایک 3x3 گھماؤ میٹرکس ہے اور $p$ ایک 3x1 ترجمہ ویکٹر ہے۔

### ڈیناویٹ ہارٹنبرگ کنونشن

ڈیناویٹ ہارٹنبرگ (DH) کنونشن روبوٹک مینیپولیٹرز پر کوآرڈینیٹ فریم وضاحت کرنے کا ایک نظامی طریقہ فراہم کرتا ہے:

1. **جڑ لینتھ** ($a_i$): $x_i$ کے ساتھ $z_{i-1}$ سے $z_i$ تک کی دوری
2. **جڑ ٹویسٹ** ($\alpha_i$): $z_{i-1}$ اور $z_i$ کے درمیان $x_i$ کے بارے میں زاویہ
3. **جڑ آفسیٹ** ($d_i$): کوآرڈینیٹ فریم $i-1$ کے اصل سے کوآرڈینیٹ فریم $i$ کے اصل تک $z_{i-1}$ کے ساتھ دوری
4. **جڑ زاویہ** ($\theta_i$): $x_{i-1}$ اور $x_i$ کے درمیان $z_{i-1}$ کے بارے میں زاویہ

### ٹرانسفارمیشن میٹرکس کا حساب

ہر جوڑی $i$ کے لیے، ٹرانسفارمیشن میٹرکس ہے:

$$A_i = \begin{bmatrix}
\cos\theta_i & -\sin\theta_i\cos\alpha_i & \sin\theta_i\sin\alpha_i & a_i\cos\theta_i \\
\sin\theta_i & \cos\theta_i\cos\alpha_i & -\cos\theta_i\sin\alpha_i & a_i\sin\theta_i \\
0 & \sin\alpha_i & \cos\alpha_i & d_i \\
0 & 0 & 0 & 1
\end{bmatrix}$$

مکمل فارورڈ کنیمیٹکس یہ ہے:
$$T_{total} = A_1 \cdot A_2 \cdot ... \cdot A_n$$

## 2.3 انورس کنیمیٹکس

انورس کنیمیٹکس مطلوبہ اینڈ ایفیکٹر پوزیشن اور اورینٹیشن حاصل کرنے کے لیے ضروری جوڑی زاویوں کا تعین کرتا ہے۔ یہ عام طور پر فارورڈ کنیمیٹکس سے زیادہ چیلنجنگ ہے۔

### تحلیلی حل

کچھ مخصوص جیومیٹرک ترتیبات والے سادہ مینیپولیٹرز کے لیے تحلیلی حل موجود ہو سکتے ہیں۔ مثال کے طور پر، 3 میل کے جوڑوں والے 6-DOF مینیپولیٹر کے بند فارم کے حل ہیں۔

### عددی حل

پیچیدہ مینیپولیٹرز کے لیے عددی طریقے اکثر استعمال کیے جاتے ہیں:
- جیکوبین کی بنیاد پر طریقے
- گریڈیئن ڈیسینٹ
- پسوڈو انورس طریقے

## 2.4 جیکوبین اور رفتار کے تعلقات

جیکوبین میٹرکس $J(q)$ جوڑی کی رفتار کو اینڈ ایفیکٹر کی رفتار سے جوڑتی ہے:

$$\dot{x} = J(q)\dot{q}$$

جہاں:
- $\dot{x}$ کارٹیزین جگہ میں اینڈ ایفیکٹر کی رفتار ویکٹر ہے
- $J(q)$ جیکوبین میٹرکس ہے جو جوڑی کے زاویوں $q$ پر منحصر ہے
- $\dot{q}$ جوڑی کی رفتار ویکٹر ہے

### جیکوبین کا حساب

جیکوبین جیومیٹرک طریقے سے حساب لگایا جا سکتا ہے:

ایک ریوولوٹ جوڑی $i$ کے لیے:
$$J_{v_i} = z_{i-1} \times (p_n - p_{i-1})$$
$$J_{\omega_i} = z_{i-1}$$

ایک پریزمیٹک جوڑی $i$ کے لیے:
$$J_{v_i} = z_{i-1}$$
$$J_{\omega_i} = 0$$

جہاں:
- $J_{v_i}$ لکیری رفتار کا حصہ ہے
- $J_{\omega_i}$ زاویائی رفتار کا حصہ ہے
- $z_{i-1}$ جوڑی $i$ کی دھوری ہے فریم $i-1$ میں
- $p_n$ اینڈ ایفیکٹر کی پوزیشن ہے
- $p_{i-1}$ جوڑی $i-1$ کی پوزیشن ہے

## 2.5 سنگولریٹیز اور مینوپلیٹبیلٹی

### سنگولریٹیز

سنگولریٹیز وہاں ہوتی ہیں جہاں جیکوبین کی رینک ختم ہو جاتی ہے، جس کا مطلب ہے کہ روبوٹ کارٹیزین جگہ میں ایک یا زیادہ ڈگری آزادی کھو دیتا ہے۔ سنگولریٹیز پر:
- روبوٹ کچھ سمت میں حرکت نہیں کر سکتا
- جوڑی کی رفتار بہت بڑی ہو سکتی ہے
- جیکوبین کا الٹا غیر متعرف ہو جاتا ہے

### مینوپلیٹبیلٹی

مینوپلیٹبیلٹی یہ پیمائش کرتی ہے کہ جیکوبین "کتنی اچھی حالت میں ہے"۔ ایک عام پیمائش یہ ہے:
$$m = \sqrt{\det(JJ^T)}$$

## 2.6 پائی تھون نافذ کاری

آئیے ایک سادہ 2-DOF ہم مستوی مینیپولیٹر کے لیے فارورڈ کنیمیٹکس کا نافذ کریں:

```python
import numpy as np
import matplotlib.pyplot as plt

def rotation_matrix(theta):
    """2D گھماؤ میٹرکس بنائیں"""
    return np.array([[np.cos(theta), -np.sin(theta)],
                     [np.sin(theta), np.cos(theta)]])

def forward_kinematics_2dof(joint_angles, link_lengths):
    """
    2-DOF ہم مستوی مینیپولیٹر کے لیے فارورڈ کنیمیٹکس کا حساب لگائیں

    پیرامیٹرز:
    joint_angles: [theta1, theta2] ریڈینز میں
    link_lengths: [l1, l2] میٹر میں

    نتیجہ:
    end_effector_pos: [x, y] اینڈ ایفیکٹر کی پوزیشن
    """
    theta1, theta2 = joint_angles
    l1, l2 = link_lengths

    # جوڑی 2 کی پوزیشن
    x2 = l1 * np.cos(theta1)
    y2 = l1 * np.sin(theta1)

    # اینڈ ایفیکٹر کی پوزیشن
    x_end = x2 + l2 * np.cos(theta1 + theta2)
    y_end = y2 + l2 * np.sin(theta1 + theta2)

    return np.array([x_end, y_end])

def jacobian_2dof(joint_angles, link_lengths):
    """
    2-DOF ہم مستوی مینیپولیٹر کے لیے جیکوبین کا حساب لگائیں
    """
    theta1, theta2 = joint_angles
    l1, l2 = link_lengths

    J = np.array([
        [-l1*np.sin(theta1) - l2*np.sin(theta1+theta2), -l2*np.sin(theta1+theta2)],
        [l1*np.cos(theta1) + l2*np.cos(theta1+theta2), l2*np.cos(theta1+theta2)]
    ])

    return J

# مثال کا استعمال
theta = [np.pi/4, np.pi/6]  # جوڑی کے زاویے
lengths = [1.0, 0.8]        # جڑ کی لمبائیاں

end_pos = forward_kinematics_2dof(theta, lengths)
jacobian = jacobian_2dof(theta, lengths)

print(f"اینڈ ایفیکٹر پوزیشن: {end_pos}")
print(f"جیکوبین میٹرکس:\n{jacobian}")
```

## 2.7 تصور کاری اور تجزیہ

```python
def plot_robot_2dof(joint_angles, link_lengths):
    """2-DOF مینیپولیٹر کنفیگریشن کو پلاٹ کریں"""
    theta1, theta2 = joint_angles
    l1, l2 = link_lengths

    # پوزیشنز کا حساب لگائیں
    joint2_x = l1 * np.cos(theta1)
    joint2_y = l1 * np.sin(theta1)

    end_x = joint2_x + l2 * np.cos(theta1 + theta2)
    end_y = joint2_y + l2 * np.sin(theta1 + theta2)

    # پلاٹ کریں
    fig, ax = plt.subplots(figsize=(8, 8))

    # جڑ کو ڈراؤ
    ax.plot([0, joint2_x], [0, joint2_y], 'b-', linewidth=5, label='جڑ 1')
    ax.plot([joint2_x, end_x], [joint2_y, end_y], 'r-', linewidth=5, label='جڑ 2')

    # جوڑی کو ڈراؤ
    ax.plot(0, 0, 'ko', markersize=10, label='بیس')
    ax.plot(joint2_x, joint2_y, 'ko', markersize=8, label='جوڑی 2')
    ax.plot(end_x, end_y, 'ro', markersize=8, label='اینڈ ایفیکٹر')

    ax.set_xlim(-2, 2)
    ax.set_ylim(-2, 2)
    ax.set_aspect('equal')
    ax.grid(True)
    ax.legend()
    ax.set_title('2-DOF ہم مستوی مینیپولیٹر')

    plt.show()

# روبوٹ کی تصور کاری
plot_robot_2dof(theta, lengths)
```
